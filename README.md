## 📌 About

**Surge** is a front-end college assignment that demonstrates a complete user
registration flow with a modern dark UI. The form walks the user through four
logical steps — account setup, personal details, address, and identity
verification — with smooth animations, instant feedback, and clear error messages.

Everything runs in a single `index.html` file. No backend, no npm, no server needed.

## ✨ Features

- **4-step wizard** — Account → Personal → Address → Identity Verify
- **Animated charge progress bar** that fills step by step
- **Real-time validation** on every field with instant hints
- **Username availability check** with simulated async lookup
- **Gmail-only email** restriction (`@gmail.com`)
- **Password strength meter** — Weak / Fair / Good / Strong
- **Canvas-drawn CAPTCHA** with noise, dots, colour variation, and refresh
- **Indian address form** with all state dropdowns
- **ID verification** — Aadhaar, PAN, Passport, Voter ID with format-specific regex
- **Shake animation** on invalid fields when trying to proceed
- **Particle canvas background** and rotating conic-gradient card border
- **Reduced-motion support** via `prefers-reduced-motion`
